# 🔍 BJ10870: 피보나치 수 5

### 1. 문제 분석
- **유형**: 재귀 (Recursion)
- **제약 조건**: $N \le 20$ (범위가 매우 작아 단순 재귀로 해결 가능)
- **시간 복잡도**: $O(2^N)$

### 2. 풀이 전략
- **Base Case**: $n=0$일 때 0, $n=1$일 때 1 반환
- **Recursive Step**: $f(n) = f(n-1) + f(n-2)$ 호출

### 3. Java 실전 팁
- `BufferedReader`를 사용하여 입력을 빠르게 처리함.
- 재귀 호출 시 스택 오버플로우를 방지하기 위해 $N$의 범위를 항상 확인해야 함.

### 4. 회고
- $N$이 커질 경우를 대비해 나중에 **Memoization(DP)** 기법을 적용해봐야겠다.
