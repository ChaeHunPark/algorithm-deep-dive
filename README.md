# 🚀 Algorithm Deep-Dive Archive
> **"원리를 파헤치고 내실을 다지는 백엔드 엔지니어의 알고리즘 공략집"**

본 레포지토리는 자료구조의 이해를 바탕으로, 주요 알고리즘 유형을 단계별로 격파하며 **기술적 근거(Time Complexity)**를 가진 해결 능력을 기르는 것을 목표로 합니다. 단순히 문제를 푸는 것에 그치지 않고, 실무 환경에서의 최적화와 확장성을 고려합니다.

---

## 📌 공략 원칙 (Strategy)
1. **정밀한 시간 복잡도 계산**: $O(N^2)$과 $O(N \log N)$의 차이를 데이터 규모에 따라 사전에 판독하여 최적의 알고리즘을 선택합니다.
2. **I/O 최적화 및 메모리 관리**: `BufferedReader`와 `StringTokenizer`를 활용하여 대용량 데이터 처리에 적합한 입출력 환경을 구축합니다.
3. **회고 기반 학습 (Persistence)**: 모든 문제 해결 후 `Analysis.md`를 통해 로직의 흐름, 시행착오, 오버플로우 방지 전략 등을 기록합니다.
